DRIFT 전환 — "edge 무용"은 방법이 아니라 데이터 탓이었다

spatial-temporal + edge 연구 성립 확정 · 2026-07-04
결론: 긴 검증 끝에 — 공업탑에는 차량 간 상호작용이 없어서 edge가 죽었던 것이고, 상호작용이 강한 DRIFT 데이터셋에선 edge가 6개 도로 전부에서 LaneChange를 유의미하게 향상(+2.0~+9.5%p)시킨다. 원 novelty(엣지 시계열)는 DRIFT에서 정당하게 성립.
